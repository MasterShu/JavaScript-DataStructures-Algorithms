== 栈

栈也是一种相当常见的数据结构。

简单的来说，你可以这么来理解栈，他就像是叠盘子一样，用的时候从上面拿，少了就往上面放，当然，这就是打个
比方，能形象的理解就好。

从上的比喻我们也能看出来，其实栈也是比较容易理解的。而且我们也可以很容易的得出这么个想法：

- 栈也是线性表
- 它操作是受限的
- 操作也仅有一进一出
- 后来者居上，也就是先到后出，后到先出
- 单个端点的操作

至此，关于栈的基本解释就结束了。相信基本上都明白一二了。
