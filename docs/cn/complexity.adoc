== 复杂度
大 O 来表示复杂度。

粗略的来分类，可以衡量算法复杂度的两个指标。时间复杂度和空间复杂度。

复杂度的量级分类。可以简单的分为多项式量级和非多项式量级。*非多项式量级* 的复杂度有： latexmath:[$O(2^n)$]
和 latexmath:[$O(n!)$]。当规模 n 越大，求解的时间会急剧增长，故这种复杂度的算法基本上处于尴尬的地位。
